# My Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated skill bars
- Project showcase
- Contact information section
- Clean and professional design

## Customization

### Personal Information
1. Open `index.html` and replace the following placeholders:
   - `[Your Name]` - Your full name
   - `[Your Profession/Tagline]` - Your job title or tagline
   - `your.email@example.com` - Your email address
   - `(123) 456-7890` - Your phone number
   - `linkedin.com/in/yourprofile` - Your LinkedIn profile URL
   - `github.com/yourusername` - Your GitHub profile URL

### Images
1. Add your profile picture as `images/profile.jpg`
2. Add project screenshots as `images/project1.jpg`, `images/project2.jpg`, etc.
3. Supported formats: JPG, PNG, GIF

### Skills
1. In `index.html`, update the skills section with your actual skills
2. Adjust the width percentages in the `style` attributes to match your proficiency levels

### Projects
1. Update the project cards with your actual projects
2. Replace the placeholder descriptions and links

### Styling
1. Open `styles.css` to customize colors, fonts, and layout
2. The color scheme uses a blue gradient - change the hex codes to match your preferences

## How to View

Simply open `index.html` in any modern web browser.

## Technologies Used

- Vanilla JavaScript
